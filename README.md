# City-Wise Wellness Center Analysis

## **Description**
This mini project focuses on analyzing operational and staffing data from a Wellness Center to identify resource gaps, performance trends, and improvement opportunities. The project covers the full analytics workflow from data preprocessing in Excel to interactive dashboarding in Power BI.

---

## **Features**
- Data cleaning, formatting, and handling missing values in Excel  
- Creation of fact & dimension tables for structured reporting  
- Interactive dashboards built in Power BI  
- Descriptive, diagnostic, predictive & prescriptive analytics  
- Insights into staffing gaps, service performance, and improvement areas  
- Clear visual storytelling with charts, KPIs, and filters  

---

## **Dataset Details**
The dataset contains:
- Staffing data  
- Operational metrics  
- Service categories  
- City/Location information  
- Performance indicators  

Data was preprocessed and transformed into **Fact and Dimension tables** for better modeling and dashboard creation.

---

## **Tools**
- **Microsoft Excel** – Data cleaning, transformation, and preprocessing  
- **Power BI** – Report modeling, DAX calculations, and interactive dashboards
