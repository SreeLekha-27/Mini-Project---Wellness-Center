# City-Wise Wellness Center Analysis
Description
This mini project focuses on analyzing operational and staffing data from a Wellness Center to identify resource gaps, performance trends, and improvement opportunities. The project covers the full analytics workflow from data preprocessing in Excel to interactive dashboarding in Power BI.

Features

Data cleaning, formatting, and handling missing values in Excel
Creation of fact & dimension tables for structured reporting
Interactive dashboards built in Power BI
Descriptive, diagnostic, predictive & prescriptive analytics
Insights into staffing gaps, service performance, and improvement areas
Clear visual storytelling with charts, KPIs, and filters

Dataset Details

The dataset contains:

Staffing data
Operational metrics
Service categories
City/Location information
Performance indicators
Data was preprocessed and transformed into Fact and Dimension tables for better modeling and dashboard creation.

Tools & Technologies

Microsoft Excel – Data cleaning, transformation, preprocessing
Power BI – Report modeling, DAX calculations, and interactive dashboards

Project Workflow

Data Collection
Preprocessing in Excel
 Remove duplicates
 Handle missing values
 Format dates, numbers, categories
 Create Fact & Dimension tables
 
Power BI Model Setup
 Import tables
 Create relationships

Dashboard Creation
 Bar, column, line & pie charts
 KPIs and cards
 Drill-down & slicers
 Bookmarks for navigation
Insight Generation & Recommendations

Dashboards & Insights

The Power BI report highlights:
Number of centers per city/category
Staffing availability vs. requirement
Performance trends by city/service
Cities needing operational improvements
Predictive indicators for resource planning

Key Learnings

Advanced Excel preprocessing
Building structured Power BI data models 
Designing interactive dashboards
Converting raw data into meaningful insights
Applying different analytical methods (descriptive, diagnostic, predictive, prescriptive)
