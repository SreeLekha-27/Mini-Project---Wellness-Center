# Mini-Project---Wellness-Center
This mini project focuses on analyzing operational and staffing data from a Wellness Center to identify resource gaps, performance trends, and improvement opportunities. The project covers the full analytics workflow from data preprocessing in Excel to interactive dashboarding in Power BI.

Project Highlights

Performed data preprocessing, including cleaning, formatting, handling missing values, and organizing data for analysis.

Designed and structured datasets using fact and dimension tables for better reporting.

Built multiple interactive dashboards to communicate insights clearly and effectively.

Applied descriptive, diagnostic, predictive, and prescriptive analytical techniques to understand patterns and support decision-making.

Identified key trends, performance indicators, and improvement opportunities based on data.

Tools & Technologies

Excel – Data cleaning & preprocessing

Power BI – Dashboard creation & visualization
